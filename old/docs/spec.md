# スマートQ&Aシステム 全体仕様書

## 1. ビジョン
あらゆる業種に対応可能な汎用型AIチャットボット。導入企業の「FAQ（知識）」をAIに学習させ、24時間365日の自動応答とVoC（顧客の声）の可視化を実現する。

## 2. コア機能
- **リファラーベース認証**: 設置ドメインからテナントを自動特定。
- **2段階AI応答 (Gemini 3.0 Flash)**:
    1. カテゴリ特定処理（動的カテゴリに対応）。
    2. 特定された知識のみを用いた回答生成（ハルシネーション抑制）。
- **管理画面**: 企業ごとのカテゴリ管理、Q&A登録、会話ログの資産化。

## 3. 技術構成
- **Framework**: Nuxt 3 (Vercel)
- **Database**: Firebase Firestore
- **AI**: Gemini 3.0 Flash
