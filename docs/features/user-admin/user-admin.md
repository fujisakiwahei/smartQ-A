# 機能詳細：利用企業向け管理画面 (Client Admin)

---

## 📊 1. ダッシュボード (分析・統計)

利用企業が指定した任意の期間（カレンダー選択）における AI ボットの稼働状況を可視化します。

### サマリーメトリクス

- **総対話数**: 指定期間内に発生した全 Q&A 実行数。
- **解決数 / 解決率（円グラフ）**: ナレッジベースに基づき AI が回答を完結させた数と割合。
- **未解決数**: AI が「答えられません」と回答した数（Q&A 追加が必要な項目）。

### ビジュアル分析

- **カテゴリ別質問頻度（円グラフ）**: どの製品やサービスに問い合わせが集中しているかを比率で表示。
- **質問ボリューム推移（棒グラフ）**: 日別・時間別のトレンド分析。

### AI 改善候補（未解決分析）

- AI が解決できなかった質問を自動抽出し、ナレッジベースへの追加候補として提示。

---

## 📁 2. カテゴリ管理

AI が質問を仕分けるための「思考の整理棚」を管理します。

### 管理項目

- **カテゴリ名**
- **説明文 (Description)**: AI がカテゴリを正確に判定するための補足情報（最大 350 文字）。具体的キーワードを含めるよう UI で誘導。

### CRUD 操作の補足

- 追加・編集・削除の変更内容は、即座に AI の分類エンジン（ステップ 1）に反映される。

---

## 📖 3. ナレッジベース管理 (Q&A Editor)

AI が回答を生成する際に唯一の正解ソースとして参照するデータを管理します。

### 管理項目

- **質問文**
- **回答文**
- **関連カテゴリ**: 1 つの Q&A に対し複数のカテゴリを紐付け可能。

### CRUD 操作の補足

- 登録・編集内容は Firestore のサブコレクションに保存され、次回の対話から回答に反映される。

---

## ⚙️ 4. ウィジェット設定

自社サイトに表示されるチャットウィジェットの外観や挙動をカスタマイズします。

### 外観カスタマイズ

- **テーマカラー選択**: 企業のブランドカラーに合わせて変更可能（blue, emerald, rose 等のプリセット）。

### 設置タグの取得

サイトに埋め込むための専用スクリプトタグを表示。

#### 設置タグ (埋め込み用ダミーコード)

※以下のコードは開発用のダミーであり、実際には企業ごとの ID が動的に発行されます。

```html
<script
  src="https://smart-qa-widget.vercel.app/widget.js"
  id="smart-qa-loader"
  data-tenant-id="DUMMY_TENANT_ID"
></script>
```

---

## 🔍 5. 対話ログビューアー

過去の全対話内容を詳細に確認し、顧客の声を分析します。

### 表示項目

- 日時、ユーザーの質問、AI の回答、判定されたカテゴリ、解決/未解決ステータス。

### フィルタ機能

- カテゴリ別、指定期間別、キーワード検索による絞り込み。
