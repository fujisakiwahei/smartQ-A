# 機能詳細：知識ベース管理 (Knowledge Management)

## 1. カテゴリ管理

- **項目**: 名称、Description（最大 350 文字）。
- **Description の活用**: AI が質問を仕分けるための辞書として機能させるため、具体的キーワードを含めるよう UI で誘導。

## 2. Q&A 管理 (CRUD)

- **登録**: 質問文、回答文、所属カテゴリ（複数選択可）。
- **データ反映**: Firestore のサブコレクション `knowledge_base` に即座に保存され、次回の AI 応答から反映される。

## 3. バルクインポート (将来拡張)

- 初期導入時に大量のデータを投入するための CSV インポート機能を検討。
