# 機能詳細：ログと分析 (Logging & Analytics)

## 1. ログデータ構造 (chat_logs)

- `tenant_id`: 識別用。
- `user_query`: ユーザーの原文。
- `ai_response`: 生成された回答。
- `detected_category`: AI が自動分類したカテゴリ名。
- `latency_ms`: 応答までにかかった時間。

## 2. ダッシュボード指標 (VoC 資産化)

- **質問ボリューム**: 日別・時間別の問い合わせ推移。
- **カテゴリ分布**: どのカテゴリの質問が多いかを円グラフで表示。
- **未解決分析**: AI が「答えられません」と回答したログを抽出し、Q&A 追加の候補として提示。

## 3. データエクスポート

- 月次レポート作成用に、ログデータを CSV 出力する機能。
