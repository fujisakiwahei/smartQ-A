# Admin UI Definitions

## 1. 開発者管理画面 (Dev Admin)
- テナント新規作成、ドメイン許可、利用状況モニタリング。

## 2. 利用企業管理画面 (Client Admin)
- Dashboard: カテゴリ別の質問頻度、解決率などの統計グラフ。
- Category Editor: 名称設定と、AIの精度を高めるための350文字説明文の編集。
- Q&A Editor: 質問・回答の登録と、複数カテゴリの紐付け。
- Log Viewer: 過去の全対話内容の確認。

## 3. ウィジェット UI
- フローティングボタン。
- メッセージウィンドウ。
- AI生成中の「考え中」ステータス表示（思考プロセスの可視化）。
